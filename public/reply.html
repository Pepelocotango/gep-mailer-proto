si no s'obre automàticament, fes clic al botó per respondre amb el teu gestor de correu:

Obrir client de correu

⚙️ Si el botó no funciona

Si uses Gmail al navegador, pots activar-lo com a gestor de correu una sola vegada i el botó funcionarà sempre.

Chrome
  1. Obre gmail.com
  2. Busca a la barra d'adreces, a la dreta, la icona dels dos rombos ◈ (o un rombe simple ◇)
  3. Clica-la → selecciona "Permet que gmail.com gestioni els links de correu (mailto)"
  4. Clica Permet
💡 Si no veus la icona, busca la icona de sliders ⊞ o un cercle amb una i 🛈 a l'esquerra de la URL
Firefox
  1. Obre gmail.com
  2. Firefox mostrarà automàticament una notificació a la part superior
  3. Selecciona "Utilitza Gmail" i clica Permet
🧭 Safari (Mac)
  1. Obre el menú Safari → Preferències → General
  2. A "App de correu predeterminada", selecciona Gmail (si apareix) o Apple Mail
  3. Si no apareix Gmail, necessitaràs instal·lar l'extensió Mailto: Gmail de l'App Store
💡 Safari és el navegador més restrictiu amb mailto: — Apple Mail com a predeterminat és la solució més senzilla
Edge
  1. Obre gmail.com
  2. A la barra d'adreces, clica la icona de permisos del lloc 🔒 (a l'esquerra de la URL)
  3. Busca "Gestors de protocols" → activa Gmail per a mailto
  4. O ves a Configuració Edge → Cookies i permisos → Gestors de protocols
📱 Mòbil (Android / iPhone)
  1. Al mòbil el botó obre directament l'app de correu predeterminada
  2. Si no funciona, comprova que tens una app de correu instal·lada (Gmail, Outlook...)
  3. A Android: Configuració → Apps → Apps predeterminades → App de correu
  4. A iPhone: Configuració → Mail → App de correu predeterminada
🦁 Brave
  1. Obre gmail.com
  2. A la barra d'adreces, clica la icona del lleó o els sliders ⊞
  3. Funciona igual que Chrome — busca "Gestors de protocols" i permet Gmail per a mailto
💡 Brave és basat en Chrome — les instruccions de Chrome funcionen aquí
🦆 DuckDuckGo
  1. DuckDuckGo (mòbil) no suporta gestors de mailto personalitzats
  2. Obre el link amb Chrome o Safari i segueix les instruccions d'aquell navegador
💡 DuckDuckGo al mòbil és molt restrictiu — recomanem usar Chrome o Firefox per a aquesta acció
🦊 LibreWolf
  1. Obre gmail.com
  2. LibreWolf és basat en Firefox però amb privacitat reforçada — pot bloquejar els gestors de protocols per defecte
  3. Ves a about:config → busca network.protocol-handler.expose.mailto → posa-ho a false
  4. Reinicia el navegador i torna a gmail.com — ara hauria d'aparèixer la notificació
💡 LibreWolf és el navegador més restrictiu d'aquesta llista — si tens problemes, Firefox és l'alternativa més compatible
🌊 Vivaldi / Opera / Arc
  1. Tots estan basats en Chrome — segueix les instruccions de Chrome de dalt
  2. La icona pot variar però la configuració de gestors de protocols és la mateixa

Un cop fet, torna aquí i clica el botó de dalt. ✅
